Text Analysis:

Here's how "Text Analysis" works.

< Server side >



1. Specified game data is collected at a designated time.

2. Nouns are extracted from the collected data, and appearance of each noun is recorded.



< Client(Browser) side >



1. Request data from DB using selected game name and time period.

2. Show top 25 most frequently occurring nouns as a bar graph.



Data is collected once every day.

Currently collecting data of: Lineage M, Girls' Frontline.

To be updated with new features.

Currently only supporting Korean text analysis.

Trending Words:

Here's how "Trending Words" work.



1. Request data to the server using search word and time period inputted.

2. Show bar graphs of two data (related search terms and terms with increased search frequency) returned.



"Trending Words" uses google trends, location is limited to Korea at the moment. Loading data may take time.

To be updated with new features.
