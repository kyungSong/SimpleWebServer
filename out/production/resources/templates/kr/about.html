서버 클라이언트

텍스트 분석

1. 매일 정해진 시간에 정해진 게임에 대한 포스팅 내용 수집.
2. 텍스트 상에서 사용된 명사를 추출하여 각 명사가 언급된 횟수 저장.

1. 게임 이름과 기간을 정한 뒤 제출 버튼을 누르면 서버측에 데이터 요청.
2. 받은 데이터중 상위 25개의 단어를 막대 그래프로 만들어 게시.

사용 해보기

데이터 채집은 매일 1회 이루어 집니다.
현재 채집되고 있는 게임: 리니지m, 소녀전선.
계속 업데이트 예정.

연관 검색어

1. 지정 검색어와 기간을 정한뒤 제출 버튼을 누르면 서버측에 데이터 요청.
2. 해당 기간동안 지정 검색어와 관련되었던 연관 검색어 바 그래프로 게시.
3. 해당 기간동안 지정 검색어와 관련되어 검색 빈도율이 높아진 검색어 바 그래프로 게시.

사용 해보기

연관 검색어 검색은 구글 트렌드를 사용하고 있으며, 지역은 한국으로 제한되어 있습니다.
검색 기간이 길면 로딩이 길어질 수 있습니다.
